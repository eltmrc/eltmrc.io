Most chat UIs train us to treat models like oracles. In practice, the useful shape is closer to a **colleague with a limited badge**: goals, tools, and a clear perimeter.

## Mental model

```
User intent → Agent plan → Tool calls → Review → Result
```

![Trust boundary between user, agent, and the world](/images/diagrams/trust.svg)

## What changes

- You stop pasting everything into one infinite thread.
- You give the agent **scoped tools**, not total access.
- You keep a human in the loop for irreversible steps.

## For Cial

Cial is built around this: sessions, tools, and identity — not a single chat box pretending to be a product.
