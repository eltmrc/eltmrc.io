Sync is expensive. I default to writing things down so other people (and future me) can join without a call.

## Default stack

1. Short written brief  
2. Async comments  
3. Meeting only when the decision is stuck  

![Idea → build → ship → learn loop](/images/diagrams/loop.svg)

## Why it scales

Async leaves artifacts. Meetings leave vibes. Artifacts compound.
